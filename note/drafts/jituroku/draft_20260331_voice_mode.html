副業でClaude Codeを使い始めて3ヶ月経ったとき、手首が痛くなった。

案件の修正指示を毎回タイピングしてたんだけど、1日の作業で40分以上をそれだけに使ってた。「このコンポーネントのスタイルを変更して。マージンは16px、ホバー時にシャドウ追加して、色は#333で〜」みたいなやつを1日に何十回も打ってたら、そりゃそうなる。

3月に /voice コマンドが追加された。声でコーディング指示できる機能。1ヶ月使い続けた結果、指示入力の時間が40分から15分に減った。その話を書く。

/voiceってどんな機能か

Claude Codeのターミナルで /voice と入力すると起動する音声入力モード。スペースキーを押しながら話して、離すと音声がテキストに変換されてClaudeに送信される。Push-to-Talk方式。

2026年3月3日に追加。Pro・Max・Team・Enterpriseプランで使える。追加料金なし。

指示入力に1日40分かかってた

導入前の副業の作業フローがこんな感じだった。

指示のループ
Claude Codeを立ち上げる → 修正指示をタイピング（1〜2分） → 実装される → 確認して次の指示を打ち込む → 繰り返し

案件によっては1セッションで20〜30回指示を出す。単純計算で30〜60分が「指示を打ち込む時間」だけで消える。

コードを書いてもらう以外の時間にコストがかかってるって気づいたのは、手首が痛くなってからだった。

1ヶ月後、15分に減った

1ヶ月使い続けた結果。

導入前
指示入力に1日40〜50分（タイピングのみ）
導入後
指示入力に1日13〜18分（音声+一部テキスト）

削減率は約65%。週5日副業してると週あたり約2時間が浮いた。月換算で8時間以上。

体感的な変化も大きくて、「タイピングしながら考える」から「考えながら話す」になった。思考をそのまま声に出すだけなので、指示の質も上がった気がする。

つまずきポイント 3つ

ここを読まずに試すと詰まるので先に書く。

① 日本語が出てこなかった

最初に試したとき、日本語が全然認識されなかった。「このコンポーネントを修正して」が「Kono component wo...」みたいな謎テキストになって笑えなかった。

デフォルトが英語認識になってる。日本語を使うには /config で変える必要がある。

/config set transcription.language japanese

これだけ。最初にやっておかないと詰まる。

② 技術用語は英語発音が正解だった

日本語設定にしても、コンポーネント名や関数名は英語発音で話した方が認識精度が上がった。

「ClickableButton コンポーネントの〜」と言うとき、「ClickableButton」の部分だけ英語で発音する感じ。日本語と英語を混在させるのが現実的な使い方だった。

③ ファイルパスはテキストで入力

/src/components/ui/Button.tsx みたいなパスを声で言おうとすると崩れる。変数名や正確なメソッド名も同様。

解決策はハイブリッド入力。声とテキストは同じセッション内で混在できる。「このファイルを修正して」は声で言いながら、ファイルパスだけテキストで打ち込む使い方が一番ストレスがなかった。

特に効果があったシーン

デバッグ相談
画面を見ながら「このエラーが出てて、原因はたぶん〜だと思うんだけど確認して」と声で状況を説明できる。タイピングより自然なコンテキストが渡せた
コードレビュー依頼
「このプルリクを全体的にレビューして、修正した方がいい箇所を優先度順に3つ教えて」みたいな長めの指示が声の方がスラスラ出た
仕様の整理
クライアントの要件メモを見ながら口頭で状況を説明して、そのままタスク分解してもらう。頭の中を整理しながら話すだけで指示が完成した

向いてないシーン

静かな環境が必要なとき（図書館・カフェ・電車内）
「実行して」「戻って」みたいな1〜2語の短い指示。声で出す手間よりタイピングの方が速い
CSSの細かい数値指定や正確な設定値（声だと誤認識しやすい。テキストで入力した方がミスが少ない）

あと声で作業してると、隣に人がいるときは少し気まずい。自宅作業のときに使うのがほぼ前提になってる。

1ヶ月使って変わったこと

指示を出すのが「タイピング作業」から「会話」になった感じがする。

副業後の手首の疲れがほとんどなくなった。これが地味に一番よかった。タイピング量が減った分、集中力を実装の確認と判断に使えるようになったのも大きい。

Claude Codeを毎日使ってる人は試してみてほしい。設定は2〜3分で終わる。最初の日本語設定（/config set transcription.language japanese）だけ忘れずに。
