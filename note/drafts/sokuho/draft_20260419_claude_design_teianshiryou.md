副業の提案資料づくり、毎回PowerPointで何時間もかけてませんか？

4/17にAnthropicが「Claude Design」っていう新しいプロダクトを出した。チャットで「こういう資料作って」って頼むと、スライドもピッチデックもワンページャーも作ってくれるやつ。

で、これを副業の提案資料案件に当てたら、明らかに景色が変わるなって直感した。

結論を先に書くと、3時間見込んでた提案書の叩き台が、15分で出てくる状態になった。

## Claude Designって何

claude.ai/design で使える。Claude Pro/Maxのサブスクに入ってれば追加料金なし。

チャット画面に「スライド10枚で、サービス紹介のピッチデック作って」みたいに投げると、見た目まで整ったスライドがプレビューで出てくる。

- 作れるもの: スライド・ピッチデック・ワンページャー・LP・マーケ画像・プロトタイプ
- エクスポート: PDF / PPTX / HTML / Canva / ZIP / Claude Code
- 編集: チャット指示・インラインコメント・直接編集・カスタムスライダー

特に「Canvaに送信」が効く。Claude Designで叩き台→Canvaで仕上げ、が一番ラク。

## 副業で何に効くか

CWの「企画書 提案書 作成」案件に全部効く。

Before: 依頼整理30分→PowerPoint1時間→デザイン1時間→修正30分=3時間
After: チャット5分→プレビュー3分→修正2分→Canva微調整5分=15分

## 実際のプロンプト

```
AI副業コンサルのサービス紹介ピッチデック、10枚で作ってほしい。
構成は、課題→提供価値→サービス内容3つ→料金→実績→導入ステップ→まとめ
色は濃紺と白、フォントは読みやすいゴシック系、余白は多めで。
```

30秒で全10枚のプレビュー完成。

## つまずき3つ

- カスタムテンプレ非対応（量産には不向き）
- Research Preview段階で生成が止まることあり
- PPTXエクスポートでレイアウト崩れ1〜2枚

最終仕上げはCanvaに送信が現実的。

## 副業で使う2パターン

1. 提案資料案件の叩き台生成（要件聞いた日にプレビューURL共有で方向性確認）
2. 自分の営業資料/プロフィール資料の量産（応募時にPDF添付で差別化）

## まとめ

Claude Designは「非デザイナーでも叩き台は作れる時代」のツール。副業で提案資料案件を受けてる人は、触らない理由がほぼない。Research Previewのうちに慣れておくと、正式版リリースで差がつく。
