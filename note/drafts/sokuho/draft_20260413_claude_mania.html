自分が使ってるAIツール、"正解"かどうか気になったことないですか？

4月11〜12日、サンフランシスコのMoscone Centerで開催されたAI業界最大級のカンファレンス「HumanX」。参加者6,500人、チケット1枚4,000ドル（約60万円）。

そこで話題を独占していたのが、自分が副業で毎日使っている Claude Code だった。

6,500人の業界人が「Claude Mania」と化した

AI検索サービスGleanのCEO、Arvind Jain氏がこの状況を 「Claude Mania」 と名付けた。「もはや宗教になっている」とまで言っている（TechCrunch 4/12報道）。

去年のHumanX（ラスベガス開催）では、OpenAIが圧倒的に主役だった。

今年は逆転していた。ある参加者は「Anthropicがmiles ahead（はるか先を行っている）」と語り、OpenAIの取締役会長Bret Taylor氏がわざわざSam Altmanのリーダーシップを擁護する場面すらあった。それくらい、空気がClaude側に傾いていた。

Anthropicの売上がOpenAIを超えた

4月7日、Anthropicの年間売上（ARR）が 300億ドル（約4.5兆円） に到達したとBloombergが報じた。OpenAIの250億ドルを初めて上回った。

2025年1月 → 10億ドル
2026年2月 → 90億ドル
2026年4月 → 300億ドル

15ヶ月で売上30倍。Claude Codeだけで2月時点の年間売上が 25億ドル（約3,750億円） を超えていた。

企業顧客の伸びもすさまじい。年間100万ドル以上使う企業が2ヶ月で500社→1,000社に倍増した。売上の8割が企業利用。もう「チャットBot」じゃなくて、業務インフラとして使われている。

副業でClaude Codeを使ってる自分が思うこと

正直、このニュースを見たとき「選んで正解だった」と思った。

自分は2月からClaude Codeで副業を始めて、2ヶ月で月14万円を稼いだ。Auto Mode、Remote Control、Monitorツール — Anthropicが出す新機能を片っ端から副業に組み込んできた。

「Claude一強」の流れが業界レベルで確定しつつあるなら、今から使い始めても全然遅くない。むしろツール選びで迷う時間がなくなった分、始めやすくなったと思う。
