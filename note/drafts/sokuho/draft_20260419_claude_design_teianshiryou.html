副業の提案資料づくり、毎回PowerPointで何時間もかけてませんか？

4/17にAnthropicが「Claude Design」っていう新しいプロダクトを出した。チャットで「こういう資料作って」って頼むと、スライドもピッチデックもワンページャーも作ってくれるやつ。

で、これを副業の提案資料案件に当てたら、明らかに景色が変わるなって直感した。

結論を先に書くと、3時間見込んでた提案書の叩き台が、15分で出てくる状態になった。

Claude Designって何

claude.ai/design で使える。Claude Pro/Maxのサブスクに入ってれば追加料金なし（Team/Enterpriseもいけるけど、Enterpriseはデフォルトでオフ設定らしい）。

チャット画面に「スライド10枚で、サービス紹介のピッチデック作って。テーマカラーは落ち着いた青系」みたいに投げる。しばらく待つと、見た目まで整ったスライドがプレビューで出てくる。

ここまでは普通の生成AIっぽい。面白いのは次。

作れるもの
スライド・ピッチデック・ワンページャー・LP・マーケ用画像・プロトタイプ
エクスポート
PDF / PPTX / HTML / Canvaへ送信 / ZIPまとめダウンロード / Claude Codeへハンドオフ
編集のしかた
チャット指示・インラインコメント・直接編集・カスタムスライダー（Claudeが勝手に「余白」「色」の調整ツマミを作ってくれる）

個人的に効くと思ったのは「Canvaに送信」機能。生成したスライドをそのままCanvaに飛ばして、あとはCanvaで細かい調整ができる。

つまり「ゼロから組むのは面倒だけど、叩き台があればCanvaで仕上げるのはできる」っていう人にちょうどハマる。自分も完全にそのタイプ。

副業で何に効くか

クラウドワークスで「企画書 提案書 作成」で検索すると、日常的に出てる案件がすぐ見つかる。

サービス紹介のピッチデック作成
起業家向け・投資家向けプレゼンの資料
営業資料作成
クライアントに持って行く用のサービス紹介資料
1ページ提案書（ワンページャー）
メールに添付する用の、A4サイズ1枚の提案資料

これ、全部Claude Designがカバーしてる範囲なんだよね。

自分が前に受けた副業だと、「スタートアップが投資家にプレゼンするピッチデック10枚」の案件が印象に残ってる。元ワークフローはこんな感じだった。

依頼内容の整理30分 → PowerPointで構造作成1時間 → デザイン整える1時間 → 修正30分 = 3時間

これをClaude Designに当てると、こう変わる。

依頼内容をチャットに投げる5分 → プレビュー確認3分 → 修正指示2分 → Canvaで微調整5分 = 15分

本当に1/10くらいの時短。これが速報で書くに値する理由。

実際に使ったプロンプト

お試しで「架空のAI副業コンサルのピッチデック」を作らせたときのプロンプトがこれ。

AI副業コンサルのサービス紹介ピッチデック、10枚で作ってほしい。
構成は、課題 → 提供価値 → サービス内容3つ → 料金 → 実績 → 導入ステップ → まとめ
色は濃紺と白、フォントは読みやすいゴシック系、余白は多めで。

これ投げて30秒くらいで、全10枚のプレビューが出てきた。色・レイアウトもかなり整ってて、「このままPDF出して送れるレベルか？」って一瞬思ったくらい。

そこから「3枚目の箇条書き3つ目を削除して、代わりにグラフの枠だけ置いて」みたいな指示で微修正できる。チャットで話してるだけで資料が整っていく感覚。

正直つまずいたところ

完璧かっていうとそうじゃない。1時間くらい触ってみて、3つ引っかかった。

カスタムテンプレが使えない
自分のブランドテンプレを登録して「これで作って」って指示するのはまだできない。「毎回同じテンプレで量産したい」タイプの案件には今のところ向かない
Research Preview段階でたまにバグる
生成の途中で止まることが2回あった。チャットをリロードすれば直ったけど、案件納期直前にやるのは怖い
PPTXエクスポートはレイアウトが崩れることがある
中身はHTMLベースで作られてるから、PPTXに出すと1〜2枚レイアウト崩れが出る。最終仕上げは「Canvaに送信→Canvaで手直し」が現実的

なので自分の今の副業ワークフローは、「Claude Designで8割を10分で作る → Canvaで2割を5分で仕上げる」の組み合わせ。これで15分で叩き台が完成する。

副業でどう使うかの2パターン

触ってみて、副業に組み込むならこの2パターンが現実的だった。

パターン1・提案資料案件の叩き台生成
クライアントから要件聞いた直後にClaude Designで叩き台を作って、その日のうちにプレビューURLを共有。「方向性これで合ってますか？」の確認を1日で済ませる
パターン2・自分の営業資料/プロフィール資料
副業で使う自己紹介スライドやポートフォリオをClaude Designで量産。案件応募時にPDFを添付して、応募文だけの人と差別化する

特にパターン2は効く。CWの応募文だけだと埋もれがちだけど、「自己紹介スライド5枚」をPDFで添付すると、提案時の見え方が変わる。しかもClaude Designで5分で作れる。

今日のまとめ

Claude Designは「非デザイナーでも叩き台は作れる時代」を押し進めるツール。副業で提案資料系の案件を受けてる人（or 受けたい人）は、触らない理由がほぼない。

Research Previewだからバグはある。でも、今の段階で触っておくと、正式リリースになったときには誰より速く使える状態になってる。

気になった人は claude.ai/design にアクセスするだけで始められる。Pro/Maxのサブスクがあれば追加料金なし。
