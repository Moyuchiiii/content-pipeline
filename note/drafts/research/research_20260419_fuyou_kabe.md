# リサーチサマリー: 月14万大学生の扶養の壁（2026年）

## 確定事実（✅ 一次・公式+複数ソース整合）

### 本人の所得税 非課税ライン
- 2025年分: 160万円まで非課税（基礎控除拡大）
- 2026年分: 178万円まで非課税（基礎控除104万円+給与所得控除74万円）
- ソース: 財務省PDF / 弥生 / taxlabor.com

### 親の扶養（特定親族特別控除・2025年新設）
- 正式名称: 特定親族特別控除
- 対象: 19歳以上23歳未満の子
- 給与収入150万円まで: 親が63万円控除（満額）
- 150万円〜188万円: 控除額が段階的に減少（逓減）
- 188万円超: 控除ゼロ
- 適用: 2025年分から
- ソース: 国税庁No.1177 / 弥生 / SmartHR / 財務省PDF

### 社会保険の扶養（2025年10月〜）
- 19歳以上23歳未満: 年収150万円未満で親の社保扶養内
- 一般（23歳以上）の130万円壁とは別扱い
- ソース: 日本年金機構 / taxlabor.com

### 副業の確定申告
- 給与以外の副業所得が年20万円超→所得税の確定申告必要
- 20万円以下でも住民税の申告は必要（区役所・市役所）
- 所得=収入-必要経費
- ソース: freee / 弥生 / 国税庁

## 🔶 信頼度高（税理士監修メディア一致）

### 月14万×12ヶ月のケーススタディ
- 年収換算: 14万×12=168万円
- 本人所得税: 178万円までOK → セーフ
- 親の扶養控除: 150〜188万ゾーン → 段階的に減る（控除額下がる）
- 社保扶養: 150万円超 → 親の社保から外れる
- 確定申告: 副業所得20万超なので必須
