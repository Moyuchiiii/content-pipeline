# 大学生が納品メッセージに1行足したら、CW継続依頼が止まらなくなった話

- 副業初月は受注4件全部単発、月収¥55,000
- 2ヶ月目に納品メッセージの最後に1行足すようにしたら継続依頼が5/7件に
- その1行をClaudeに書かせるプロンプト + 3パターンテンプレ公開
