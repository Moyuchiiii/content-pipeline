AI副業始めたいけど、ChatGPTとClaude、結局どっちに課金すればいいの？

自分もこれ、めちゃくちゃ悩んだ。

ネットで調べると「ChatGPTが万能」って記事もあれば「Claudeの方がすごい」って記事もある。しかも比較記事のほとんどが「ベンチマークスコアがどうこう」みたいなエンジニア向けの話で、「副業で稼ぎたい自分はどっちを使えばいいの？」に答えてくれない。

結論から言うと、両方に課金して2ヶ月使い倒した結果、場面ごとに使い分けるのが一番稼げた。

今は月14万の副業収入があるけど、その内訳を振り返ると「ここはChatGPT」「ここはClaude」って明確に分かれてた。この記事では、その使い分けを全部書く。

先に結論だけ見たい人へ

忙しい人向けに、自分の使い分けルールを3行で書く。

応募文・提案書・メール
Claude（ブラウザ版 claude.ai）。日本語がそのまま使えるレベルで出てくる
リサーチ・画像生成・データ分析
ChatGPT。画像を作れる（DALL-Eっていう画像生成AI入り）のと、データ分析が強い
コードを書く・ファイルを作る・自動化する
Claude Code（ターミナル版）。PCの中で直接作業してくれるのが圧倒的

以下、具体的にどの場面でどっちを使ったか、全部書いていく。

そもそもChatGPTとClaudeって何が違うの

「AIチャットでしょ？同じじゃないの？」って思ってた。自分も最初そうだった。

ざっくり言うと、ChatGPTは「いろんなことが1つでできる万能ツール」、Claudeは「文章とコードに特化した職人ツール」みたいな感じ。

ChatGPT（OpenAI）
画像を作れる（DALL-Eっていう画像生成AI内蔵）。データ分析もできる（Excelアップロード→グラフ化が一発）。Web検索もできる。とにかく「1つで全部やりたい人」向け
Claude（Anthropic）
日本語の文章がめちゃくちゃ自然。長い文章を読み込ませても崩れない。Web検索もできる。Claude Codeを使うとPC上でファイルを直接編集してくれる。「文章を書く仕事」「コードを書く仕事」をやるならこっち

料金はどっちも月額約3,000円（$20）。無料版もあるけど、副業で使うなら有料版じゃないとすぐ制限に引っかかる。

応募文を書く → Claude一択だった

クラウドワークスで案件に応募するとき、最初はChatGPTで応募文を書いてた。でも出てくる文章が「です・ます」の連打で、どう見ても「AI感」がすごかった。

同じ指示をClaudeに出したら、日本語の質が全然違った。敬語のバランスとか、「相手が読みたい順番」で書いてくれる感覚がある。

ChatGPTの応募文
「貴社の業務効率化に貢献できると確信しております。私はAIツールを活用した業務自動化の経験があり、迅速かつ丁寧に対応いたします。」← 誰が書いても同じ
Claudeの応募文
「案件詳細を拝見しました。月次レポートのExcel整理+グラフ作成とのことですが、以前同じような案件で30分かかっていた作業を5分に短縮した経験があります。具体的には〜」← 案件内容に踏み込んでる

Claudeの方が「この人ちゃんと案件読んでるな」って思わせる文章になる。実際、Claude で応募文を書き始めてから受注率が上がった。

画像生成・データ分析 → ChatGPTの勝ち

案件を受けた後、クライアントに見せるための画像やグラフが必要になる場面がある。ここはChatGPTの独壇場。

ChatGPTにはDALL-E（ダリ）っていう画像生成AIが組み込まれてて、「こんなイメージの図を作って」って言えばその場で画像が出てくる。Claudeは画像を「見る」ことはできるけど「作る」ことはできない。

データ分析もChatGPTが強い。Excelファイルをアップロードして「グラフにして」「傾向を分析して」って頼むと、コードを書かなくても自動で処理してくれる。クライアントへの報告書を作るときにめちゃくちゃ助かった。

Web検索は実はどっちもできる。ただChatGPTの方がスムーズに検索結果を会話に取り込んでくれるから、調べ物のテンポはChatGPTの方が良い。

ChatGPTが強い場面
画像生成（サムネ・図解・イメージ図）、データ分析（Excelグラフ化・傾向分析）、調べ物のテンポ

コードを書く・ファイルを作る → Claude Codeが別次元

副業で「Excelマクロ作って」「Webサイトのここ直して」みたいな案件をやるとき、Claude Code（ターミナルで動くClaude）を使ってる。

Claude Codeの何がすごいかっていうと、PCの中のファイルを直接触ってくれる。ChatGPTはチャット画面にコードを表示するだけだから、自分でコピペして貼り付ける必要がある。Claude Codeは「このファイルを直して」って言うと勝手にファイルを開いて書き換えてくれる。

自分の場合、業務自動化スクリプトの案件で使うと、ChatGPTだと2〜3時間かかるところがClaude Codeだと30分で終わる。コピペの手間がないだけでこんなに変わる。

Claude Codeが向いてる作業
Webサイト修正、スクリプト作成、ファイル整理の自動化、納品物の一括生成。「PC上で何かを作る」作業は全部これ

ただしClaude Codeはターミナル（黒い画面）で動くから、最初だけちょっと戸惑う。とはいえ、コマンドは1つ覚えるだけでいい。インストールして claude と打つだけで起動する。

レポート・資料作成 → 組み合わせが最強だった

クライアントに出すレポートや報告書を作るとき、自分はChatGPTとClaudeを組み合わせてる。

流れはこう。

  1. ChatGPTでWeb検索してデータを集める（「〇〇業界の市場規模は？」とか）
  2. 集めたデータをClaudeに渡して、レポートの文章を書かせる
  3. 必要ならClaude Codeでファイルとして出力する

ChatGPTだけで全部やると「調べた内容をそのまま貼っただけ」みたいなレポートになりがち。Claudeに渡すと「読みやすい構成」に整えてくれるから、クライアントの反応が良くなった。

失敗した使い方も正直に書く

最初の1ヶ月は、使い分けなんて考えずに全部ChatGPTでやってた。

失敗1「応募文がテンプレ臭くて受注できなかった」

ChatGPTで応募文を量産してた時期がある。効率は良かったけど、3週間で受注ゼロだった。後から見返すと、どの応募文もほぼ同じトーンで「AI感」がすごかった。Claudeに変えてから1週間で初受注が来た。

失敗2「Claudeで画像を作ろうとして詰んだ」

クライアントに「イメージ図を1枚つけてほしい」と言われたとき、いつもの流れでClaudeに頼んだ。でもClaudeは画像を作れない。慌ててChatGPTに切り替えてDALL-Eで生成した。「文章はClaude、画像はChatGPT」って最初から決めておけば焦らなかった。

失敗3「Claude Codeを使わずにコピペ作業してた」

最初はChatGPTのチャット画面でコードを生成→手動でファイルにコピペ→エラーが出たらまたチャットに戻る、を繰り返してた。Claude Codeを使い始めてからこの往復がゼロになった。なんで最初から使わなかったのか後悔してる。

結局、副業で月14万稼いでる自分の使い分けルール

2ヶ月間いろいろ試した結果、こう落ち着いた。

Claude（ブラウザ版 claude.ai）を使う場面
応募文・提案書・クライアントへのメール・記事の下書き・長い文章の要約。「日本語で何かを書く」作業は全部こっち
ChatGPTを使う場面
画像生成・データ分析・グラフ作成・リサーチの下調べ。「画像を作る」「データを見せる」作業はこっち
Claude Code（ターミナル版）を使う場面
Webサイト修正・スクリプト作成・ファイル整理・自動化。「PCの中で何かを作る」作業はこっち

副業を始めたばかりで「1つだけ選ぶなら？」と聞かれたら、最初はChatGPT。無料版でも十分試せるし、調べ物も画像も全部1つでできるから。

でも副業で「文章を書く案件」が増えてきたら、Claudeに課金した方がいい。応募文の質が変わるし、受注率に直結する。

そしてコードを書く案件をやるなら、Claude Codeは必須。コピペ作業がなくなるだけで1案件あたりの作業時間が半分以下になる。

「どっちも課金するのはもったいない」と思う人へ

自分も最初そう思ってた。月6,000円（ChatGPT $20 + Claude $20）は学生には痛い出費だった。

でも冷静に計算すると、1案件で¥5,000〜¥20,000もらえる副業で、作業効率が2倍になるツールに月6,000円。1案件受注すれば元が取れる。

実際、Claude に課金してから応募文の質が上がって受注率が上がったし、Claude Codeで作業時間が半分になったから受けられる案件数も増えた。月6,000円のツール代で月14万稼げてるなら、投資対効果としてはバグってる。

もし「まず1つだけ」なら、ChatGPTから始めて、文章を書く仕事が増えたらClaudeを足す。この順番がいいと思う。
