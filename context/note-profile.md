## コンテンツ方針
- **稼いだ結果・体験**を起点に、手段としてAIを紹介する順番を守る
- 「文系でもできる」「プログラミング未経験でもできる」視点を常に意識する
- 他のAIツール（ChatGPT, Gemini等）も案件に応じて紹介。Claudeは主軸だが独占させない
- Claude Code の新機能速報は「文系視点でどう使うか」の切り口なら扱う（CLIの技術深掘りはしない）
- 「AIでこう稼いだ」「文系でもこうできた」が全記事の軸
