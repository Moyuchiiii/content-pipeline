# 文体サンプル

既存記事から抽出した文体の特徴。記事生成時はこのトーンに合わせること。

## トーンの特徴

- 一人称: 「自分」「僕」
- 語尾: 「〜みたいな」「〜ってやつ」「〜なんだよね」等のカジュアルな口語
- 読者への語りかけ: 「〜じゃないですか」「〜だと思う」
- 技術説明: 噛み砕いて説明。専門用語は初出時に補足
- 正直さ: 「正直に書いておくと」「完璧じゃないこともある」
- 具体性: Before/After、数字（30分→5分、毎日2時間）

## サンプル文

### 導入部の書き方
> 朝起きて、スマホでニュースアプリをダラダラ見る時間、意外と長くないですか。自分もそうだった。

> 情報系の大学生をやってると、コードを書く以外にも地味な作業が多い。データ整理、ファイルのリネーム、調べものをまとめてレポートにする、みたいなやつ。

### 技術説明の書き方
> RSSフィードっていうのは、Webサイトが新着記事を配信する仕組み。多くのニュースサイトが対応していて、プログラムから記事一覧を取得するのにぴったり。

> Anthropicが出してるCLIツールで、ターミナルからClaudeを呼び出してコードを書いたり、ファイルを操作したりできる。

### まとめ部分の書き方
> やってることはシンプルで、「RSSフィードを取得してテキストに吐き出す」だけ。でも、これだけで朝の時間の使い方がガラッと変わった。

> 「AIに仕事させる」って聞くとすごいことやってる感あるけど、実際は地味な作業を代わりにやってもらってるだけ。

## NGパターン（これは使わない）
- 「〜と言えるでしょう」「〜ではないでしょうか」
- 「まず最初に」「重要なのは」の連発
- 絵文字の多用
- 「■」「【】」付きの見出し的箇条書きブロック
- 括弧（）での補足説明の多用

### タイトル設計ルール

**優先して使うパターン（スキが取れる型）:**
- 「〜してみてわかった、〇〇のこと」（体験実録型。大学生の強み）
- 「実は〜は〜だった」（逆張り意外性型。梶谷で2,042スキ実績あり）
- 「〜したら〜が変わった」（Before/After型）
- 数字付き（「〇つのポイント」「〇日間で〜」）

**使わないパターン（競合と被る・moyuchiに合わない）:**
- パイプ（｜）区切り二部タイトル → FabyΔの型そのまま。被る
- 「〜完全ガイド」「〜超完全ガイド」→ 梶谷と被る
- 「〜が〜を突きつけた」「残酷な〜」等の重厚語彙 → ビジネス層向けで大学生らしくない

### 冒頭設計ルール

**推奨（moyuchiの武器を活かす）:**
```
体験型（最推奨）: 「先週〇〇をやってみたら〇〇分→〇〇分になった。」+ 転換
共感型: 「〜ってやりにくくないですか。自分もずっとそう思ってた。」+ 転換
数値型（口語版）: 「今月のClaude Code代、2.3万円だった。」+ 短い事実 + 転換
```

**使わない冒頭:**
- 「〇年〇月〇日、〜が発表された。」（FabyΔ型。ニュース系クリエイターの型）
- 「〜な方は少なくありません」（梶谷型。コンサル的で大学生らしくない）

### noteエディタ装飾ルール（競合分析から追加）

**使う:**
- backtick（`〜`）: ファイル名・コマンド・ツール名の強調（`CLAUDE.md`等）
- 「▼」記号: 参考リンク・関連記事の誘導
- 番号付きリスト: HowTo系の手順（h2に番号を振る）
- カギカッコ（「」）: キーワード・概念の強調、引用

**使わない（競合分析で確認済み）:**
- 引用blockquote → 名言・引用は「」括弧内で処理
- コードブロック多用 → 1〜2個に抑える
- 区切り線多用 → 記事がブツ切りに見える

### 有料ライン設計ルール

- **無料ゾーン**: 体験エピソード＋「なぜこれが重要か」の問い
- **有料ゾーン**: 具体的な手順・設定ファイル・プロンプトテンプレート
- **最強の購買動機**: 「Claude Codeに渡すだけで動く」形式のファイル特典（梶谷が実証済み）
- **価格帯**: ¥500〜¥1,500（梶谷の¥1,980〜と差別化。学生向け）

### moyuchiだけの差別化ポジション

競合2名（FabyΔ=投資家、梶谷=ビジネスコンサル）が持っていない強み:
1. **大学生の等身大視点** — 失敗・試行錯誤・コスト感覚の正直な開示
2. **同世代との近さ** — 「自分もスタートラインは同じ」という読者との距離感
3. **機動力** — 最新ツールを触ってすぐ記事化できるスピード
4. **学習プロセスの可視化** — プロには書けない「最初から全然わからなかった話」
