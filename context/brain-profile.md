## 文体ガイド（Brain プロフィール用）

- note のカジュアル口語（「〜なんだよね」「〜じゃん」）は **プロフィールでは控えめ** にする
  - Brain 読者はレビュー前提で「買うに値するか」を判断するため、ちょっと信頼感寄りに
  - 商品本文では note と同じカジュアル文体で OK
- **一人称**: 「わたし」（ひらがな）で統一（「僕」「俺」「自分」は使わない、note・X と同じ。詳細は `persona-hyui.md` 参照）
- **絵文字**: 0〜1 個まで。「↓」等の誘導記号は OK
- **箇条書き**: プロフ文では使わない（自然文で流す）

## 避けるべき表現（ガイドライン違反ブラックリスト）

- 「誰でも」「必ず」「絶対に」「100%」
- 「稼げる」「稼げます」（主語なしで読者を主語にしてしまうパターン）
- 「知らないと損」「人生が変わる」
- 「不労所得」「寝てても稼げる」
- 「○○が稼げる方法」（読者全員達成前提の表現）
- 「低学歴でも」等の差別的修飾

→ **「私が」「自分が」を主語に**する。「達成しました」「やってみました」「記録です」で締める。
