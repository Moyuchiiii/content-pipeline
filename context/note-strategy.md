# 収益化戦略ノウハウ

## 差別化軸（競合分析より）

「副業ゼロの**文系**大学生がAIだけで2ヶ月で月14万を達成した実録」が核心の差別化軸。
FabyΔの「NVIDIA 6億円」と同等のポジション。全記事タイトル・プロフィール・メンバーシップ説明に一貫させる。
**フレーミングの順番**: 稼いだ結果 → 失敗・試行錯誤 → 手段（Claude/AI）の順で語る。ツール紹介から入らない。

## 記事タイプ優先順位

実録 > ノウハウ > 速報

- 実録: スキ転換率30%（全記事最高値）。「実際に稼いだ」体験がnote読者に最も刺さる
- ノウハウ: AIで量産可能。週2〜3本ペースで安定的に出せる
- 速報: トレンド乗りには使うが収益の柱にしない

## 成長フェーズ設計（FabyΔ参考）

メンバーシップは開始から運用中。

- フェーズ1（〜3ヶ月）: 週3〜4本ペース（実録週1 + ノウハウ週2〜3）。メンバーシップ運用中
- フェーズ2（3〜6ヶ月）: 毎日投稿に増量。メンバー数を増やす
- フェーズ3（6ヶ月〜）: 「完全ガイド」1本を次のClaudeブレイクに合わせて投下。転換点を狙う

## 転換点記事（完全ガイド）の設計

FabyΔは「Claude Code超完全ガイド」1本でスキ2,026を獲得。Hyuiも同様の転換点記事を仕込む。
ただしタイトルは **結果ファースト型** にする（「〜で月○万になった全手順」「副業ゼロから○ヶ月で変わった話」等）。

- タイミング: Claudeの次の大型アップデートリリース時 or 実績が積み上がったタイミング
- 分量: 通常記事の3〜5倍
- 内容: 非エンジニア視点で「副業を始めて稼げるようになるまでの全記録」+ Claudeを使った具体的な手順
- 価格: 有料（1,000〜2,000円）

## メンバーシップ戦略

- 名称: 副業ゼロから月14万になった大学生の全記録
- 価格: 月500円

## ジャンル選定の3要素（スコアリング基準）

記事のトピックを選定する際、以下の3つの掛け算でスコアリングする。

### 1. 市場の大きさ（1〜5点）
- 5点: X/noteで関連キーワードの発信者が多数、フォロワー数万人規模のアカウントが複数
- 4点: 一定の発信者がいる、関心層は広い
- 3点: ニッチだが確実に需要がある
- 2点: 関心層が限定的
- 1点: ほとんど需要が見込めない

### 2. 悩みの深さ（1〜5点）— HARM法則ベース
- 5点: Money（稼ぎたい）に直結。お金を払ってでも知りたい
- 4点: Ambition（キャリア・スキルアップ）に直結
- 3点: HARMに間接的に関連
- 2点: 「知ってると便利」レベル
- 1点: 純粋な好奇心・雑学

### 3. 悩みの緊急度（1〜5点）
- 5点: 今すぐ知らないと損する（新機能リリース直後、期間限定）
- 4点: 近い将来必要になる
- 3点: いつか必要になるかも
- 2点: 急ぎではない
- 1点: 完全に任意

### 4. 差別化可能性（1〜5点）
- 5点: 誰も書いてない＋自分の実体験で語れる
- 4点: 競合は少ない＋独自の切り口がある
- 3点: 競合はいるが差別化できる角度がある
- 2点: 競合が多く差別化が難しい
- 1点: 既に良質な記事が飽和

### 5. 過去実績との相性（1〜5点）
- 5点: 過去にスキ数が多かったテーマと同系統
- 4点: 過去記事と関連性が高い（シリーズ化できる）
- 3点: 新しいテーマだが読者層は重なる
- 2点: 読者層が少しずれる
- 1点: 全く新しい方向

### 6. 非エンジニア理解度（1〜5点）
- 5点: エンジニア知識ゼロでも読める。「自分にもできそう」と思わせる内容
- 4点: 専門用語は出るが、都度説明があれば理解できる
- 3点: 多少の技術知識があれば理解できる
- 2点: エンジニア向けの内容が多く、非エンジニアには難しい
- 1点: 技術者専門の内容。副業初心者には届かない

## 有料記事の無料/有料エリア設計

### 無料エリア（冒頭30〜40%）
- 記事の価値が伝わる導入
- 「何が得られるか」のリスト
- 実績・Before/Afterの一部
- 無料エリアだけでも「役に立った」と感じさせる

### 有料→無料境界に入れるもの
- 「この先で手に入るもの」リスト（3〜5項目）
- 「なぜ有料なのか」の理由（外注相場との比較等）

### 有料エリア（残り60〜70%）
- 実際のコード/プロンプト全文
- 具体的な数字・設定ファイル
- テンプレート・チェックリスト
- カスタマイズ方法・応用例

## メインコンテンツ戦略（Claude / Anthropic 特化）

### コンセプト
「副業でAIを使って稼ぎたいならこの人」— 非エンジニア・副業初心者（男性中心）が最初に辿り着くポジションを取る。ブラウザ版Claude / ChatGPT が主軸ツール、Claude Code は補助的に登場。文系大学生が作るが、男性読者も自分ごと化できるトーンで発信。

### 扱うトピック
- **副業・稼ぎ方**（AI・Claudeを使った実際の受注・納品・収益の記録） ← 最優先
- Claude Code / Claude の実務活用テクニック（副業に直結する内容）
- 「非エンジニアでもできた」体験・失敗談
- クラウドワークス・ココナラでの案件獲得ノウハウ（AI活用込み）
- Claude Codeの新機能・アップデート速報（副業に使える文脈で紹介）
- Claude vs 他ツール比較 ← 「どちらが稼ぎやすいか」視点で書く

### 扱わないトピック
- ChatGPT / Gemini / その他AIツール単体の話題
- AI全般のニュース（副業・稼ぎ方に絡まない限り）
- プログラミング一般論（副業・Claude活用と絡めない限り）
- エンジニア向けの技術深掘り（副業初心者が置いていかれる内容）

### 収益モデル
- Claude新機能リリース → 「最強の使い方」を速攻で有料500円
- 副業・稼ぎ方ノウハウ完全ガイド（Claude活用手順込み）→ 有料1,000〜2,000円（テンプレート・設定ファイル付き）
- 入門記事・比較記事 → 無料（集客・ブランディング）
- 実録・ノウハウは週1でストック型コンテンツとして蓄積
- メンバーシップ月500円 → 有料記事読み放題で継続収益を確保
