#### 文体・スタイル特徴

| 要素 | 詳細 |
|------|------|
| 1人称 | なし（主語省略が多い） |
| 口調 | 断定的・センセーショナル・ビジネス実用 |
| フック | 【速報】【⚡️速報】【🔥朗報】【超朗報】などの括弧見出し |
| 長さ | 中程度〜長め。スレッドで続きを展開するパターン多数 |
| 感情表現 | 「Claude強すぎる…」「面白いのは」「良い。」など短く締める |
| インパクトワード | 「なんと」「遂に」「革命的」「激震」「吹き飛んだ」 |
| 数字 | 必ず入れる（+13%、70言語、3000億ドル等） |
| スレッド | 重要ニュースは「スレッドで解説👇」で展開 |
| ハッシュタグ | ほぼ使わない（稀にギブアウェイ時のみ） |

#### 画像の使い方

- AI新機能のデモスクリーンショット（動画GIFも含む）
- 英語原文ツイートのスクショ（情報源として添付）
- 公式ロゴ系画像はほぼ使わない
- 自前スクショ＋シンプルな箇条書きテキスト画像の組み合わせ

#### 引用RTのスタイル

英語の公式発表ツイートや有名アカウントのツイートを引用RTし、日本語で要点を先に3〜5行にまとめるパターンが主流。引用コメントは【速報】で始まり、「詳細↓」「スレッドで解説👇」で締めることが多い。

#### 「稼ぐ」訴求

直接的な「稼げる」訴求はなし。ビジネスへの影響（株価・業界崩壊）を伝えることで間接的に危機感・チャンス感を演出。

### チャエン観察から読み解く「刺さるテーマ」

1. **具体的な数字×Before/After**：「3時間→15分」「月30時間削減」など定量化した変化
2. **業界への影響**：「○○業界に激震」「○○が不要になる」的な刺激ワード
3. **「これだけで良い」系**：シンプルで再現性の高い手順
4. **感情先行→詳細後展開**：1行で感情を動かし、スレッドで詳細を説明

### hyui版の応用

- 「文系の私でもできた」というコンテキストを毎回省略せず添える
- エンジニアではない＝再現性が高い、の訴求軸
- note/Brain記事の裏側を「作業実況」形式で出す（透明性→信頼→購買）
- Claudeの新機能を「大学生の副業に使えるか？」視点でコメント

## 画像戦略の現実解

### チャエン等の観察結果

| インフルエンサー | 主な画像スタイル |
|---|---|
| チャエン | 英語ツイートのスクショ＋動画GIF。公式ロゴは稀 |
| 木内翔大 | Canva風の図解・インフォグラフィック（背景色付き） |
| すぐる | デモ動画のスクショ・GIF。シンプルなキャプチャ中心 |
| ChatGPT研究所 | 機能説明図・スクショ |

### 結論：公式ロゴ路線の現実

**チャエンは公式ロゴをほとんど使わない。スクショ＋テキストが主流。**

hyuiの「Claudeロゴ+アイコン」シンプル路線は：
- 初期（フォロワー少数）段階では目立たない
- 信頼感・公式感の演出に使えるが、単品での拡散力は低い
- スクショ（実際に使っている証拠）との組み合わせが現実的

**推奨する画像戦略（現実的な2段階）：**

1. **Phase 1（現在〜フォロワー1000人）**：画像ありの比率は50%以下でOK。スクショ重視。Before/Afterなら数字だけのシンプルテキスト画像。作業スクショが最も「実在感」を出せる
2. **Phase 2（1000人〜）**：Canvaで背景色+白テキストの「刺さるコピー画像」を週2〜3枚。ブランドカラーを固定してシリーズ化

## ハッシュタグ戦略

チャエン・すぐるともにハッシュタグはほぼ使わない。
理由：Xのおすすめアルゴリズムはハッシュタグよりもエンゲージメント率を重視するため。

**hyui推奨方針**: 基本的にハッシュタグなし。例外として「#Claude」「#AI副業」を月1〜2本のピン投稿に使う程度。

## まとめ：hyui向けX運用設計要点

1. **引用RT最優先アカウント**: @claudeai / @ClaudeDevs（毎日チェック）→ @masahirochaen（週3〜4回チェック）
2. **引用RTコメントの型**: 「速報感+個人の視点+副業への応用」の3点セット
3. **日常4本の黄金比**: 作業Before/After(1)＋AIニュース所感(1)＋副業の裏側(1)＋問いかけ(1)
4. **画像**: 作業スクショが最強。Claudeロゴ単体は単品使用を避けスクショと組み合わせる
5. **文体**: 「文系大学生の自分でもできた」を毎回ゆるく入れる。断定語尾（「できる」より「できた」）
6. **ハッシュタグ**: 原則なし
