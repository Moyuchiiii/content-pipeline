# /content-engine

1つのアンカー資産（note記事草稿・体験メモ・調査結果）から、プラットフォームネイティブなコンテンツを複数フォーマットで一括生成する。

## 使い方

```
/content-engine {note草稿パス}             # note記事からX・カード等を全展開
/content-engine {テキストや体験のメモ}      # 体験メモから全フォーマット展開
/content-engine {草稿パス} --formats x     # Xのみ生成（フォーマット指定）
```

---

## Phase 0: コンテキスト読み込み

```
context/voice-samples.md     — Hyui の声・トーン
context/writing-rules.md     — 禁止フレーズ・AI感排除ルール
context/x-hook-formulas.md   — Xフック型10種
```

---

## Phase 1: アンカー資産の読み込み

アンカー資産（note記事・体験メモ等）を読み込み、以下を把握する:

- 記事タイプ（速報/実録/ノウハウ）
- 核心メッセージ（一番伝えたいこと1行）
- Before/After・数字・発見（あれば）
- 読者が「これは知らなかった」と思うポイント

---

## Phase 2: アトミックアイデア抽出

アンカー資産から「各プラットフォームで独立して成立するアイデア」を **3〜7個** 抽出する。

**抽出基準:**
- 記事全体を読まなくても理解できる独立した主張
- Before/After・数字・驚き・発見のいずれかを含む
- 同じ内容の言い換えはカウントしない（重複NG）

**出力フォーマット（作業メモ）:**
```
アトミックアイデア一覧:
① [内容] — [何が伝わるか]
② [内容] — [何が伝わるか]
...

選定理由付きで「Xで最も伸びそうな2〜3個」を選ぶ
```

---

## Phase 3: プラットフォーム別コンテンツ生成

### 3-A: Xスレッド（5〜8ツイート）

`context/x-hook-formulas.md` からフック型を1つ選び宣言してから書く。

**構成:**
```
ツイート1（フック）: 選択したフック型で1行
ツイート2〜6（中身）: 1ツイート1アイデア。箇条書きNG。文章で
ツイート7（まとめ+誘導）: 「詳しくはnoteに」+ URL
```

**声ルール（content-engineの核心）:**
- 1ツイート = 1つの明確なアイデアのみ
- フックは要約ではなく「引き」で書く（「〇〇してみた結果」「これ知らない人多すぎる」）
- リンクは本文ではなくリプライに
- 具体的スローガンNG → 「効率化できる」より「40分が5分になった」

### 3-B: 単発ツイート（1〜2本）

note記事の「一番驚いた点」を1ツイートで完結させる。

```
[事実ファースト or 驚きの1行]
[自分の視点・感想（1〜2行）]
[詳しくはnote → {URL}]（リプライ推奨）
```

### 3-C: ノウハウカード（画像テキスト・任意）

カード化すると保存・拡散されやすいポイントを1つ選んで、140字以内のキャプションテキストを出力する。

```
[テーマ: {キーワード1語}]
[1行フック: {テーマの核心15字以内}]
[サブテキスト: {補足1行}]
→ Canvaまたは Gemini ImageFX でカード化（`gemini_prompt_guide.md` を参照）
```

---

## Phase 4: CTAの整合チェック

全フォーマットを並べて確認:

- [ ] 各フォーマットで同じコピーをそのまま貼り付けていないか
- [ ] Xはnoteへ、noteはフォローへ、という導線が自然か
- [ ] 宣伝感が出ていないか（体験・実況ベースか）

---

## Phase 5: 出力

### ファイル保存

```
x/drafts/threads/draft_{日付}_{トピック}.md
x/drafts/singles/draft_{日付}_{トピック}.md
```

### コンソール出力

```
✅ content-engine 完了
アンカー資産: {ファイル名}
抽出アイデア数: {N}個
---
生成フォーマット:
  Xスレッド: {ツイート数}本 → {パス}
  単発ツイート: {本数} → {パス}
  カードテキスト: {枚数}（あれば）
```

---

## 鉄則

1. **フォームに合わせる。同じコピーを貼らない。**
2. **フックは要約より引き。**
3. **1投稿 = 1つの明確なアイデア。**
4. **スローガンより具体的な数字・体験。**
5. **CTAは小さく明確に。**
