# 単発ツイート草稿 — Remote Control副業実録

---

## 単発ツイート A（型4: 驚き・発見型）★推奨

図書館でPCをカバンに入れたまま、スマホだけで副業案件が進んだ。

以前なら「図書館行ったから今日は無理か」で終わってた。

Claude Code Remote Control、スマホで承認するだけでいい。

（詳しくはリプライに↓）

---

## 単発ツイート B（型1: 数字型）

Claude Code、承認のたびにPCの前に戻ってた。週4〜5時間。

Remote Controlを使ったらその時間がほぼゼロになった。

移動中にスマホで承認するだけ。セットアップも1分で終わった。

---

## 単発ツイート C（型9: 実況型）

バイト帰りの電車で副業案件の仕上げをやってる。

PCじゃなくてスマホで承認だけ押してるんだけど、家に着いたらほぼ完成してる予定。

Claude Code Remote Control、もう戻れない。
