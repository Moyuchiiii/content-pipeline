# 単発ツイート候補

## 候補1（速報型）
Anthropicがnpmに512,000行のコードをうっかり公開してた（本物のニュース）

4/1だからエイプリルフールっぽいけど VentureBeat も The Register も普通に報道してる。バージョン2.1.88に59.8MBのデバッグファイルが同梱されてしまった。

## 候補2（機能紹介型）
Claude Codeのリークコードで見つかった2つの未来機能

KAIROS: 入力なしで常時稼働。15秒ごとに作業を確認してアクション
ULTRAPLAN: Opus 4.6に30分考えさせて設計書を作る

どっちも「Claudeに全部投げて放置する」の究極形。副業ユーザーとして来たら即試す。
