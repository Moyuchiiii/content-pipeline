# X告知ドラフト — メイン商品タイムセール

作成日: 2026-04-12
ステータス: 草稿（送信はユーザーが行う）

---

## パターンA（数字インパクト型）

```
副業ゼロから2ヶ月で月14万になった話、全手順をnoteにまとめました。

・最初の2週間は0件受注
・何を変えたか
・実際に使った応募文テンプレも全公開

今だけ1,480円でタイムセール中です（通常2,000円）
```

リプライに: リンク貼る

---

## パターンB（Before/After型）

```
2週間応募して0件 → 1ヶ月目5.5万 → 2ヶ月目14万

何を変えたかを全部書いた記事、タイムセール中です

応募文のテンプレートもそのまま使えるやつを載せてます
```

リプライに: リンク貼る

---

## パターンC（問いかけ型）

```
クラウドワークス、応募しても全然受注できないって人へ

自分も最初の2週間は0件でした
変えたのは応募文の書き方だけです

その方法を全部まとめた記事、今だけ1,480円のセール中
```

リプライに: リンク貼る

---

## 使い方メモ

- 外部リンクは本文ではなくリプライに置く（本文直埋めでリーチ低下するため）
- 画像は記事のスクリーンショット or 収益実績のスクショが効果的
- ハッシュタグは入れなくてOK（宣伝感が出るため）
- 送信はユーザーが行う
